Get Started

One-time setup · Models download on first use

📺 Video Walkthrough

System Requirements

  • ✓ Windows 10/11 (preferred platform)
  • ✓ Blender 5.2 or later
  • ✓ NVIDIA GPU with 6 GB+ VRAM
  • ✓ CUDA 12.4
  • ✓ 20 GB+ free disk space
  • ~ Limited support for Linux

Before You Begin

1. Install Git (must be on your system PATH).

2. Download Blender 5.2+ and unzip it into your Documents folder.

3. Download Pallaidium .ZIP.

Tip: shorten the Blender folder name — long paths can cause unzip failures on Windows.

  1. Run as Administrator: right-click blender.exe → "Run as Administrator". Required for write permissions on Windows.
  2. Install the add-on: Preferences → Add-ons → Install → select the downloaded Pallaidium ZIP → enable it.
  3. Install Dependencies: in Add-on Preferences click Install Dependencies and wait for it to finish.
  4. Open the studio: restart your computer, launch Blender as Admin, open the Video Sequence Editor → Sidebar (N) → Generative AI.
First run: the chosen model downloads automatically (5–10 GB). The screen may appear frozen during this. This is normal — do not close Blender.
View on GitHub

If any modules are missing after install, use blender_pip to install them manually.

Extend the Studio

Companion Add-ons

Pallaidium is designed to work alongside the broader Blender ecosystem. These add-ons cover the rest of the production pipeline.

Narrative

Blender Screenwriter

Write screenplays, add image prompts per scene, and convert everything to timed text strips for batch generation.
Prompting

GPT4BLENDER

Use a local LLM (GPT4ALL) to generate creative image prompts and story ideas directly inside Blender.
Workflow

Text to Strip

Convert text from the Blender Text Editor into VSE strips, ready to use as batch prompts for generation.
Editing

Subtitle Editor

Navigate, edit, import, and export text strips in the VSE. Pairs with Hviske Subtitles for a full captioning workflow.
Inpainting

VSE Masking Tools

Create precise masks on top of clips in the Sequencer for use with Pallaidium's inpaint and img2img workflows.
Conversion

Add Rendered Strips

Render any strip type to a movie file and drop it back on the timeline — essential for feeding 3D renders into Pallaidium.
Diagnostics

Pallaidium Module Checker

Verifies that all required Python modules are installed and reports which models are ready to run error-free.
