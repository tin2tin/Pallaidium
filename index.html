Installation Protocol

System Requirements

  • ✓ Windows 10/11 (Preferred)
  • ✓ NVIDIA GPU (6GB+ VRAM)
  • ✓ CUDA 12.4
  • ✓ 20GB+ HDD Space

Pre-Flight Check

1. Install Git (add to PATH).

2. Download Blender 4.5.x.
⚠ Do NOT use Blender 5.0.

  1. Run as Admin: Right-click blender.exe & select "Run as Administrator".
  2. Install Add-on: Preferences > Add-ons > Install > Select Pallaidium ZIP.
  3. CRITICAL: In Add-on Preferences, click Uninstall Dependencies first to clear conflicts.
  4. Restart Blender (Always as Administrator).
  5. Install Deps: In Add-on Preferences, click Install Dependencies.
  6. Launch: Restart Computer. Open VSE Sidebar (N) > Generative AI.
NOTE: First run downloads 5-10GB of models. The screen may freeze. This is normal behavior.
